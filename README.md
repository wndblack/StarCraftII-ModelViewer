StarCraftII-ModelViewer
=======================

StarCraftII Model Viewer is a tool to show latest Sc2 Model 

Currently show:
- Geometry
- Bones
- Animation
- Particle Emitters
- Ribbons

Setting up:

This is a Directx&cpu version,it can compile with dxsdk ,cximage ,and wxWidgets;Directx&gpu version has not complete
